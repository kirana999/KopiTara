# Coffetara.id - Website Katalog & Storytelling Kopi Tara

Selamat datang di repositori resmi **Coffetara.id**. Website ini dikembangkan sebagai platform digital untuk Kopi Tara yang berfokus pada katalog produk, narasi perjalanan brand, dan pemberdayaan mitra.

## 🚀 Fitur Utama

* **Landing Page:** Identitas brand dengan pesan filosofis "Digiling Langsung di Tempat".
* **Tentang Kami:** Narasi *storytelling* perjalanan transformasi bisnis dari furnitur kayu jati menjadi bisnis kopi yang berkelanjutan.
* **Katalog Produk:** Daftar produk kopi Nusantara yang terbagi berdasarkan daerah asal dengan filter daerah yang interaktif.
* **Sistem Varian & Harga:** Pengaturan harga dinamis berdasarkan ukuran kemasan (100g, 250g, dll).
* **Integrasi Penjualan:** Tombol "Beli Sekarang" yang mengarahkan pelanggan langsung ke link Shopee untuk setiap varian produk.
* **Desain Responsif:** Kompatibilitas tampilan di perangkat desktop dan mobile.

## 📂 Struktur Proyek

Berikut adalah peta menyeluruh dari struktur direktori proyek ini:

```text
Coffetara.id/
├── public/                # Aset statis (gambar, logo, favicon)
│   └── images/            # Kumpulan gambar produk (kopi-1.png, kopi-2.png, dll)
├── src/
│   ├── assets/            # Gambar pendukung kebun kopi & petani
│   ├── components/        # Komponen UI modular
│   │   ├── Navbar.js      # Navigasi website
│   │   └── Footer.js      # Footer website
│   ├── pages/             # Halaman utama website
│   │   ├── Home.js        # Halaman Beranda
│   │   ├── About.js       # Halaman Tentang Kami
│   │   └── Catalog.js     # Halaman Katalog Produk
│   ├── styles/            # Kumpulan file CSS
│   │   └── global.css     # Styling global, tema warna, dan layout
│   ├── App.js             # Konfigurasi Routing (React Router)
│   └── index.js           # Titik masuk aplikasi (Entry point)
├── .gitignore             # File yang diabaikan oleh Git
├── package.json           # Konfigurasi dependensi proyek
└── README.md              # Dokumentasi ini

```
